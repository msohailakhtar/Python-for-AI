#  Chapter 01 – Python Fundamentals

Welcome to **Chapter 01 – Python Fundamentals** of the **Python for Artificial Intelligence** course.

This chapter provides the essential programming foundation required for Artificial Intelligence, Machine Learning, Data Science, and Software Development.

---

#  Learning Objectives

By completing this chapter, you will learn how to:

- Understand Python syntax
- Work with variables and data types
- Use operators and expressions
- Accept user input and display output
- Apply type casting
- Make decisions using conditional statements
- Solve repetitive tasks using loops
- Create reusable functions
- Process and manipulate strings
