# Common Mistakes to Avoid

- Incorrect indentation.
- Confusing `=` with `==`.
- Forgetting to convert user input.
- Infinite `while` loops.
- Missing `return` statements.
- Using invalid string indexes.
- Repeating code instead of writing functions.
