# Project Requirements

The program should display the following menu:

```
========================================
      STUDENT MANAGEMENT SYSTEM
========================================

1. Add Student

2. Display Student Information

3. Calculate Grade

4. Display Result

5. Exit
```

---

# Functional Requirements

## Option 1 – Add Student

Accept the following information:

- Student Name
- Roll Number
- Age
- Department
- Marks in Python
- Marks in Mathematics
- Marks in Statistics

---

## Option 2 – Display Student Information

Display

```
Student Name

Roll Number

Age

Department
```

---

## Option 3 – Calculate Grade

Calculate

```
Total Marks

Average Marks
```

Assign Grade

```
90+  → A

80+  → B

70+  → C

60+  → D

Below 60 → F
```

---

## Option 4 – Display Final Result

Display

```
=================================

Student Report

=================================

Name

Roll Number

Department

Python

Mathematics

Statistics

Total

Average

Grade

Result

=================================
```

Result

```
Pass

Fail
```

A student passes if the average marks are **50 or above**.

---

## Option 5 – Exit

Display

```
Thank you for using the Student Management System.

Goodbye!
```
